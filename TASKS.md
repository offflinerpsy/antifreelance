# Roadmap проекта BotStudio

Этот файл содержит пошаговый план разработки сайта-визитки и Базы знаний для агентства BotStudio.

## Этап 1: Базовая структура и дизайн (Выполнено)
- [x] Инициализация проекта на Astro.
- [x] Настройка глобальных стилей (CSS Variables: `--black`, `--white`, `--orange`, `--lime`).
- [x] Создание компонентов: Header, Hero, Cases (Кому подходит), Reviews, CTA.
- [x] Настройка мобильного меню и адаптивной сетки (CSS Grid).
- [x] Интеграция GSAP для анимаций появления (`.reveal`).
- [x] Кастомная анимация кнопок (`@keyframes auto-press`).

## Этап 2: База знаний (Блог) (Выполнено)
- [x] Настройка Astro Content Collections (`src/content/config.ts`) с Zod-схемой.
- [x] Создание главной страницы Базы знаний (`src/pages/knowledge/index.astro`).
- [x] Создание динамического роутинга для статей (`src/pages/knowledge/[slug].astro`).
- [x] Стилизация Markdown-контента (заголовки, списки, цитаты, изображения).
- [x] Добавление тестовой статьи.
