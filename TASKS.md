# Roadmap проекта BotStudio

Этот файл содержит пошаговый план разработки сайта-визитки и Базы знаний для агентства BotStudio.

## Этап 1: Базовая структура и дизайн (Выполнено)
- [x] Инициализация проекта на Astro.
- [x] Настройка глобальных стилей (CSS Variables: `--black`, `--white`, `--orange`, `--lime`).
- [x] Создание компонентов: Header, Hero, Cases (Кому подходит), Reviews, CTA.
- [x] Настройка мобильного меню и адаптивной сетки (CSS Grid).
