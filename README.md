# BotStudio (ex-antifreelance)

## Описание проекта
Сайт-визитка и блог (База знаний) для агентства **BotStudio**, занимающегося разработкой умных Telegram-ботов с локальным AI для бизнеса.

**Цель проекта:** Привлечение клиентов (B2B), демонстрация кейсов (Портфолио) и публикация экспертных статей (База знаний) для SEO и прогрева аудитории.

## Архитектура системы
Проект построен на современном стеке для статических сайтов:
- **Фреймворк:** [Astro](https://astro.build/) (v5+)
- **Стилизация:** Vanilla CSS с использованием CSS Variables для темизации (цвета: `--black`, `--white`, `--orange`, `--lime`).
- **Анимации:** [GSAP](https://gsap.com/) (ScrollTrigger) для плавного появления элементов (`.reveal`) и сложных скролл-анимаций в портфолио.
- **Контент:** Astro Content Collections (Markdown + Zod) для управления статьями Базы знаний.

## Структура проекта
- `src/pages/` — Роутинг (Главная страница, База знаний).
- `src/content/knowledge/` — Markdown-файлы статей Базы знаний.
- `src/components/` — Переиспользуемые UI-компоненты (Header, Hero, Cases, Portfolio, Reviews, CTA).
- `src/layouts/` — Базовые HTML-шаблоны.
- `src/styles/` — Глобальные стили и переменные.

## Как работать с этим репозиторием (Для ИИ-ассистентов)
Этот репозиторий подготовлен для автономной работы ИИ-агентов.
- В файле `AGENT_INSTRUCTIONS.md` находятся **строгие инструкции по архитектуре и правилам работы**. Обязательно к прочтению перед внесением изменений!
- В файле `TASKS.md` находится декомпозиция задач (Roadmap).
- Все задачи и баги трекаются через **GitHub Issues**.
